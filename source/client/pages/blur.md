# Размытие

> UIBlur

Размытие текстовых и блочных значений

```javascript
import UIBlur from '@xaota/ui/blur.js';
```

<ui-html>
  <p>
    Строка простого текста с
    <ui-blur>
      приватными
    </ui-blur>
    данными
  </p>
</ui-html>

### Атрибуты
> `disabled` - отключает размытие

<ui-html>
  <p>
    Строка простого текста с
    <ui-blur disabled>
      открытыми
    </ui-blur>
    данными
  </p>
</ui-html>
